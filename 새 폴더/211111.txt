flex 속성은 부모 요소에 반드시 적용해야.
flex-wrap : 가로 정렬, flex 내부 줄바꿈 지원, reverse로 역행
flex-flow : direction & row 축약.
justify-content : row의 시작점. flow에서 row-reverse로 반대 가능.
align-items : cross-axis 기준으로 flex-item 정렬
>>>> order : html 코드 수정 없이 속성값 지정만으로 재정렬 가능. flex의 강점. 절댓값 기준이라 음수도 가능.
